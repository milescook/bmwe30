## Overview

This repo is a collection of diagrams, wiring plans and 3d printable models. The objective is converting a BMW E30 to use speeduino with a digital dashboard using ts-dash. This uses the original Motronic 1.3 harness to de-risk the conversion. The only wiring modification needed so far, is a new version of the injector sub-loom (later harnesses use a sub loom with a connector), and the TPS wires either needs an adapter or re-wire to work with an E36 variable TPS.

You can find a load of shorts and videos on progress here [https://www.youtube.com/cookracinguk](https://www.youtube.com/cookracinguk)
